RemoveBOM v1.0.0
================

This program remove BOM header from UTF-8 files. Just drag and drop file(s) or directory to list and all files are checked and when
have BOM header, the header is remove.

You need .NET Framework 2 to run this application (http://www.microsoft.com/downloads/details.aspx?familyid=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=en).
