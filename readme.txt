RemoveBOM © Jakub Trmota, 2012 (http://forrest79.net)


This program remove BOM header from UTF-8 files.


HOW TO USE:
===========
Just drag and drop file(s) or directory to list and all files are checked and when have BOM header, the header is remove.


HISTORY
=======
1.0.0 [2008-11-21] - First public version
2.0.0 [2009-10-08] - Add option for list only BOM files and for testing files for having BOM


REQUIREMENTS
============
You need .NET Framework 2 to run this application (http://www.microsoft.com/downloads/details.aspx?familyid=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=en).
